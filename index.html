🖥 Windows 版文件包
下载后将文件解压到 PPSSPP 材质目录：
C:\Users\你的用户名\Documents\PPSSPP\PSP\TEXTURES\ULJM05093\

文件包内含：
  • textures.ini — 材质配置文件（放在 ULJM05093 根目录）
  • CAUGE/speed1.png — 你选择的表盘图片
📱 Android 版文件包
下载后将文件放到 PPSSPP 安卓版材质目录：
/Android/data/org.ppsspp.ppsspp/files/PSP/TEXTURES/ULJM05093/

注意：Android 11 及以上版本受 Scoped Storage 限制，可能需要使用文件管理器（如 MT管理器）访问该路径。

文件包内含：
  • textures.ini — 材质配置文件（放在 ULJM05093/new/ 目录）
  • CAUGE/speed1.png — 你选择的表盘图片

📖 使用帮助

按照以下步骤操作，轻松替换 Initial D Street Stage 的转速表盘。

📋 操作步骤

  1. 选择表盘 — 在「表盘」页面选择一个预制表盘，或上传你自己制作的表盘图片（推荐 PNG 格式，透明背景）。
  2. 下载文件包 — 前往「下载」页面，选择你的平台（Windows / Android），点击下载按钮获取 ZIP 文件包。
  3. 解压文件 — 将下载的 ZIP 文件解压。
  4. 放入材质目录 — 将解压后的 textures.ini 和 CAUGE 文件夹放入 PPSSPP 的材质目录中（详见下方路径说明）。
  5. 启动游戏 — 在 PPSSPP 模拟器中启动 Initial D Street Stage，即可看到替换后的转速表盘。

📂 材质目录路径

Windows 默认路径：

C:\Users\你的用户名\Documents\PPSSPP\PSP\TEXTURES\ULJM05093\

Android 默认路径（Android 11+）：

/Android/data/org.ppsspp.ppsspp/files/PSP/TEXTURES/ULJM05093/new/

如果你修改过 PPSSPP 的配置路径，请将文件放到你自定义的材质目录中。 关键是确保 textures.ini 位于 ULJM05093（或 ULJM05093/new/）目录下， CAUGE 文件夹也在同一目录中。

⚠️ 注意事项

  • 如果你已有 textures.ini 文件且包含其他自定义材质，建议手动合并而非直接覆盖。本工具生成的 ini 文件只包含转速表盘的替换配置。
  • 表盘图片会被重命名为 speed1.png 并放在 CAUGE 文件夹中。
  • 如果替换后没有效果，请检查 PPSSPP 设置中是否启用了「替换纹理」（Replace textures）选项。
  • 自定义表盘推荐使用 PNG 格式、正方形尺寸、透明背景，以获得最佳效果。

🔧 textures.ini 说明

本工具生成的 textures.ini 内容如下：

[options]
version = 1
hash = quick

[hashes]
00000000000000007ca9cd35 = CAUGE/speed1.png
00000000000000005e3eb2a4 = CAUGE/speed1.png
000000000000000077e3f8ea = CAUGE/speed1.png
099c7a90f177f24d262df61a = CAUGE/speed1.png

其中 4 个哈希值对应游戏中同一个转速表盘的不同变量地址，全部指向你选择的表盘图片。
